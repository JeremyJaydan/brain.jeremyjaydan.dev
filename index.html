• Usage:
  • Start typing in the subject input to create a subject.
  • Type & hit enter in the new node input to attach a new node to the selected subject.
  • Single click a node to select that node.
  • Double click a node to delete the node (does not delete child nodes).
  • Select a node then click the link button, then select another node in order to link those two nodes together.
  • Click the mode button to switch between branch/road modes.
